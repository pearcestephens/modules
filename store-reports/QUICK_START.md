## For Store Managers (Using Mobile UI)

### 1. **Access the App**
```
URL: https://staff.vapeshed.co.nz/modules/store-reports/views/mobile/create-report.php
Device: Mobile phone or tablet (works best on mobile!)
```

### 2. **Create a Report**
```
1. Select your store from dropdown
2. Go through checklist sections
3. For each item:
   - Click PASS ‚úì if okay
   - Click FAIL ‚úó if not okay
   - Click N/A ‚àí if not applicable
4. Add photos/voice memos as needed
5. Click "Save Draft" or "Submit Report"
```

### 3. **Take Photos**
```
1. Click camera icon under an item
2. Tap capture button to take photo
3. Use left/right buttons to switch cameras
4. Photo auto-uploads and appears in the app
```

### 4. **Record Voice Memo**
```
1. Click microphone icon under an item
2. Speak into your phone microphone
3. Click OK when done
4. AI automatically transcribes to text
```

### 5. **Chat with AI Assistant**
```
1. Tap purple robot icon (bottom right)
2. Ask questions like:
   - "What should I check in the refrigeration section?"
   - "How do I handle expired products?"
   - "What are critical safety items?"
3. Get instant answers from AI
4. Click suggested chips for common questions
```

### 6. **Save Your Work**
```
- App auto-saves every 30 seconds while editing
- Click "Save Draft" to manually save
- Come back anytime to continue
```

### 7. **Submit Report**
```
1. Complete your checklist
2. Click "Submit Report" button
3. Report sent to admin
4. AI analysis happens automatically
5. Check back later for feedback
```

## Common Questions

### Q: What if I don't finish a report?
A: Auto-save keeps your work. Come back anytime to resume.

### Q: Can I take multiple photos for one item?
A: Yes! Click camera multiple times for the same item.

### Q: What if I'm offline?
A: App caches your work. It will sync when you're back online (Phase 2).

### Q: How long do AI responses take?
A: Typically 2-3 seconds. You'll see a typing indicator.

### Q: Can I edit a submitted report?
A: Not yet (Phase 2). For now, submit means final.

### Q: Who can see my photos?
A: Only store managers and admins at your store can see them.

### Q: Where does my data go?
A: All data stored in secure CIS database + MCP Hub for AI analysis.

## Troubleshooting

### Camera not working?
```
1. Check phone gives camera permission
2. Try switching cameras (left/right buttons)
3. Close other camera apps
4. Restart your phone
```

### Photos won't upload?
```
1. Check internet connection (WiFi or 4G)
2. Make sure file is under 10MB
3. Try again in a few seconds
4. Refresh the page
```

### AI chat not responding?
```
1. Check internet connection
2. Wait 5 seconds for response
3. Try a simpler question
4. Refresh and try again
```

### Can't find my draft?
```
1. Make sure you're selecting the correct store
2. Drafts auto-delete after 30 days
3. Contact admin if lost
```
