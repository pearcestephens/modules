### ThemeEngine Usage
```php
$theme = ThemeEngine::getInstance();
$theme->switchTheme('professional-dark');
echo $theme->styles();  // Load CSS
echo $theme->scripts(); // Load JS
$theme->render('view', ['data' => $value]);
```

### MockData Usage
```php
$stores = MockData::getStores();
$products = MockData::getProducts();
$orders = MockData::getOrders();
$metrics = MockData::getMetrics();
```

### Adding New Layouts
1. Create PHP file in `themes/professional-dark/views/`
2. Use existing components (`.cis-*` classes)
3. Integrate MockData
4. Add to theme's `index.php` switcher

### Creating New Themes
1. Copy `professional-dark` folder structure
2. Update `theme.json` with new colors
3. Customize CSS variables in `main.css`
4. Adapt component styles as needed
5. Test all 3 layouts
