# Archived Prototypes, Templates, and Half-Finished Tools

This directory stores deprecated, experimental, or half-finished assets consolidated from:
- /modules/cis-themes
- /modules/admin-ui

Policy
- Nothing runs from here in production.
- Keep for reference or future retrofit only.
- All files moved retain original path under archived/ for traceability.

Structure
- archived/
  - {original-module}/{original-subpaths}/...

Retention
- 12 months default, then prune unless explicitly whitelisted.

Restoration
- Move file back to its maintained module path via a PR with justification.
