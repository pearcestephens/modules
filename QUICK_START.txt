📁 STRUCTURE
modules/
├── _base/              Infrastructure (DO NOT ROUTE DIRECTLY)
│   ├── assets/
│   │   ├── css/
│   │   │   └── cis-modules.css
│   │   └── js/
│   ├── lib/
│   │   └── Controller/
│   └── views/
│       └── layouts/
│           └── master.php    ← USE THIS FOR ALL MODULES
│
└── your_module/        Your feature module
    ├── controllers/
    ├── views/
    ├── lib/
    └── assets/

🎯 CREATE A NEW MODULE

1. Copy template:
   cp -r modules/_module_template modules/your_module

2. Update controller:
   vim modules/your_module/controllers/HomeController.php

3. Use master layout:
   <?php
   $pageTitle = 'Your Module';
   $content = 'Hello World!';
   include __DIR__ . '/../../_base/views/layouts/master.php';

4. Test:
   curl https://staff.vapeshed.co.nz/modules/your_module/
